5° aula(Recuperando dados no HTML)
=> dentro de views.py podemos criar variaveis e chamar com {{nome da variavel}} dentro do arquivo htmal
=> No caso do laço for dentro de views, pegar a variavel e coloca-la dentro do 'context'
=> Exemplo:
=> def about(request):
    frutas = ['Banana', 'Maça', 'Uva', 'Abacate']
    context = {
        'frutas': frutas  # Certifique-se de que o nome da variável bate com o template
    }
    return render(request, 'about.html', context)

}

    text-align: center;

=> importar o css no html.
=> <link rel="stylesheet" href="{% static 'css/style.css' %}">
=> Dentro ainda do html importar abaixo do !doctype html {% load static %}
